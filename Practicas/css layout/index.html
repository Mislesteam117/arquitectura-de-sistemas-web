¿Qué es un CSS Layout?

Un CSS Layout es la estructura visual de una página web creada usando CSS. Permite organizar el contenido en áreas como encabezado, navegación, contenido principal y pie de página.

En este ejemplo se utiliza CSS Grid para distribuir las secciones de la página de manera moderna y responsiva.
